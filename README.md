continue문과 레이블이 있는 continue문은 루프 본체의 남은 부분의 실행을 생략하고 다음 반복으로 진행

%d (10진수로 출력) / %o(8진수) / %x(16진수 az소문자) / %X(16진수 A부터Z 대문자) / %C(문자로서 출력) / %f(소수점 형식으로 출력) ex) %0.5=소숫점 다섯 번째 자리까지 출력 
%s(문자열로 출력)
